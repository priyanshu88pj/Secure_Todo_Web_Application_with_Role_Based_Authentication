Secure Todo Web Application

This is a full-stack MERN (MongoDB, Express, React, Node.js) application where users can register, log in, and manage their todos. The project includes role-based authentication, allowing admin users to access additional features.

Features:
-User Registration & Login (JWT Authentication)
-Create, Edit, Delete Todos
-Todo Categories (Urgent / Non-Urgent)
-MongoDB database integration


Tech Stack:

Frontend: React, Axios, CSS
Backend: Node.js, Express.js, MongoDB, Mongoose
Authentication: JWT, bcryptjs
